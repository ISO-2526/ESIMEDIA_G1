# Carpeta de Archivos de Audio

Esta carpeta almacena los archivos de audio estáticos del frontend.

## Formato de archivos soportados:
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- AAC (.aac)

## Instrucciones:

Coloca aquí los archivos de audio que quieras usar como respaldo o para desarrollo.
En producción, los audios se servirán desde el backend en `http://localhost:8080/audio/`.

## Nota:

Los archivos deben sincronizarse con el backend para que estén disponibles durante la reproducción.
